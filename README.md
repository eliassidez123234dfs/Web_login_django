# RED | Plataforma Web de Personalización y Estampación de Prendas

## Descripción

RED es una empresa dedicada al diseño, estampación y personalización de camisetas y otras prendas textiles. Actualmente, la empresa busca fortalecer su presencia en la web mediante el desarrollo de una plataforma que permita a sus clientes personalizar y adquirir productos de forma sencilla e interactiva.

La aplicación web permitirá que los usuarios diseñen sus propias prendas utilizando un editor de personalización, donde podrán modificar colores, estampados e imágenes y visualizar el resultado antes de realizar la compra. Además, podrán explorar un catálogo de diseños creados por los expertos de RED y personalizarlos según sus preferencias.

Una vez finalizado el diseño, el usuario podrá realizar el pago correspondiente y la empresa recibirá la información necesaria para fabricar la prenda con las especificaciones seleccionadas. Finalmente, el producto será enviado al domicilio del cliente.

## Objetivo

Desarrollar una aplicación web que permita a RED fortalecer su presencia digital, ofreciendo una plataforma moderna para la personalización, visualización y compra de prendas textiles, mejorando la experiencia del usuario y optimizando el proceso de venta.

## Instalación

1. Clonar el repositorio.
2. Crear un entorno virtual.
3. Instalar dependencias.
4. Ejecutar las migraciones.
5. Iniciar el servidor.

### Comandos:

git clone <url-del-repositorio>

cd nombre_proyecto

python -m venv venv

### Windows
venv\Scripts\activate

### Linux/Mac
source venv/bin/activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
